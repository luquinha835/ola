# ola
apps

## Executar localmente

Requisitos: Python 3.8+ e `pip`.

Instalar dependências:

```bash
pip install -r requirements.txt
```

Rodar o app:

```bash
python app.py
```

Rodar testes:

```bash
pytest -q
```

**Link do App**

Se você criou o serviço no Render com o nome `ola-web`, o app ficará disponível em:

https://ola-web.onrender.com

Se o serviço tiver outro nome, substitua pelo domínio fornecido pelo Render.

Para abrir rapidamente (no host com `$BROWSER` configurado), use o script `open_app.sh` criado no repositório.
